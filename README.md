# DWG Optimizer

Мощное приложение с графическим интерфейсом для комплексной автоматизации работы с архитектурными и инженерными САПР-файлами. Инструмент позволяет проводить пакетную замену текста, глубокую очистку исходников, пакетную печать и экстремальное сжатие файлов DWG и PDF.

## 🚀 Основной функционал

* **Пакетная замена текста:** Умный поиск и замена текста (TEXT, MTEXT, а также скрытые атрибуты внутри блоков и штампов) во всех пространствах чертежа с сохранением оригинального форматирования (шрифтов, цветов).
* **Глубокая очистка DWG:** Автоматическое выполнение команды `_WBLOCK` (ПБЛОК) во временные файлы и гарантированное отключение всех видов рамок (`_IMAGEFRAME`, `_PDFFRAME`, `_WIPEOUTFRAME`, `_XCLIPFRAME`, `_FRAME`) с поддержкой любых языковых локализаций AutoCAD.
* **Умная пакетная печать:** Автоматический экспорт актуальных листов AutoCAD в формат PDF с использованием плоттера `DWG To PDF.pc3`. Алгоритм умеет распознавать и корректно именовать специфические листы (например, с приставкой "СП").
* **Работа с PDF (без Adobe):** Склейка разрозненных листов одного чертежа в единый документ и максимальное сжатие (удаление неиспользуемых объектов, сжатие потоков данных и очистка синтаксиса).
* **Безопасная остановка (Graceful Shutdown):** Возможность в любой момент безопасно прервать процесс обработки. Программа корректно завершит текущую операцию, закроет COM-соединение и удалит временные файлы без риска повредить исходные чертежи или оставить зависшие процессы `acad.exe`.
* **Продвинутый GUI:** Современный интерфейс с поддержкой Drag-and-Drop (перетаскивание файлов и папок из Проводника Windows), динамической шкалой прогресса и информационными всплывающими подсказками (Tooltips).

## 🛠️ Стек технологий

* **Python 3:** Основной язык логики и связки модулей.
* **win32com.client (pywin32):** Прямое взаимодействие с COM-интерфейсом (ActiveX API) приложения AutoCAD.
* **PyMuPDF (fitz):** Сверхбыстрая работа с PDF-документами (склейка и сжатие).
* **Tkinter & TkinterDnD2:** Построение графического интерфейса пользователя с нативной поддержкой перетаскивания файлов в ОС Windows.
* **Регулярные выражения (re):** Для парсинга и очистки скрытых тегов MTEXT-форматирования AutoCAD.

## 💻 Сборка проекта

Для компиляции исходного кода в независимый `.exe` файл используется умный скрипт **`build.bat`**. 
Скрипт автоматически:
1. Вычищает кэш от старых сборок.
2. Создает изолированное виртуальное окружение (`venv`) и устанавливает зависимости из `requirements.txt`.
3. Собирает программу через `PyInstaller` (в режиме `--onefile` и `--noconsole`).
4. Автоматически подхватывает скрытые библиотеки (`tkinterdnd2`, `win32com`) и "вшивает" пользовательскую иконку `icon.ico`.
5. Переносит готовый исполняемый файл в корневую директорию проекта.
