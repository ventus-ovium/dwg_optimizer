# DWG Optimizer

Приложение с графическим интерфейсом для автоматизации работы с архитектурными САПР-файлами.Инструмент позволяет проводить пакетное сжатие файлов DWG и PDF.

## 🚀 Функционал
* **Глубокая очистка DWG:** Автоматическое выполнение команды `_WBLOCK` и отключение рамок (`IMAGEFRAME`, `PDFFRAME`, `WIPEOUTFRAME`, `XCLIPFRAME`, `FRAME`).
* **Пакетная печать:** Автоматический экспорт листов AutoCAD в формат PDF с использованием встроенного плоттера `DWG To PDF.pc3`.
* **Работа с PDF:** Склейка документов и их максимальное сжатие (удаление неиспользуемых объектов и очистка синтаксиса) без использования Adobe Acrobat.
* **Удобный GUI:** Интерфейс с поддержкой Drag-and-Drop для быстрого добавления папок и отдельных файлов в очередь обработки[cite: 1].

## 🛠️ Стек технологий
* **Python:** Основной язык логики[cite: 1].
* **win32com.client:** Для прямого взаимодействия с API приложения AutoCAD[cite: 1].
* **PyMuPDF (fitz):** Для работы со сжатием PDF-файлов[cite: 1].
* **Tkinter & TkinterDnD2:** Для построения графического интерфейса пользователя[cite: 1].

## 💻 Сборка проекта
Для сборки готового `.exe` файла используется скрипт `build.bat`[cite: 1]. Он автоматически создает виртуальное окружение, устанавливает зависимости из `requirements.txt` и компилирует программу через `pyinstaller` с параметрами `--onefile` и `--noconsole`[cite: 1].
